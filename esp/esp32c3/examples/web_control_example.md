# Web控制示例

本示例展示了如何通过Web界面控制ESP32-C3开发板。

## 功能概述

- LED开关控制
- 系统状态监控
- 实时数据显示

## 使用步骤

### 1. 配置WiFi

打开 `src/main.cpp` 文件，修改以下配置：

```cpp
const char* ssid = "YourWiFiSSID";        // 替换为你的WiFi名称
const char* password = "YourWiFiPassword";  // 替换为你的WiFi密码
```

### 2. 编译和上传

```bash
# 编译项目
pio run

# 上传到开发板
pio run --target upload
```

### 3. 访问Web界面

1. 打开串口监视器（波特率：115200）
2. 观察输出的IP地址
3. 在浏览器中输入该IP地址
4. 你将看到控制界面

### 4. 控制LED

通过Web界面可以：
- 开启LED
- 关闭LED
- 切换LED状态

## 故障排除

### 无法连接WiFi

- 检查WiFi名称和密码
- 确保路由器支持2.4GHz
- 检查信号强度

### Web页面无法访问

- 确认WiFi连接成功
- 检查IP地址是否正确
- 尝试使用IP地址直接访问

### LED无响应

- 检查LED连接
- 确认GPIO配置正确
- 查看串口错误信息
